# CitasApp — Sistema de Gestión Clínica

CitasApp es una solución web desarrollada bajo el patrón de arquitectura **ASP.NET Core MVC**. El sistema está diseñado para optimizar el control de agendas médicas.

## Objetivo

El propósito fundamental de este proyecto es analizar y comprender la implementación de vistas arquitectónicas en el desarrollo de software, evaluando los *trade-offs* técnicos implicados en la separación de responsabilidades y la gestión de persistencia ligera.

## Tecnologías Utilizadas

* **Framework:** ASP.NET Core MVC (.NET 10)
* **Vistas:** Razor Pages (HTML5 / CSS3 Personalizado)
* **Diseño UI:** Bootstrap (Enfoque Mobile-First / Responsivo)
* **Persistencia:** Archivos JSON (Estructuras de datos planas)
